## 🚀 一鍵部署到Railway

1. 點擊上方 "Deploy on Railway" 按鈕
2. 連接您的GitHub賬號
3. 選擇此倉庫：`tie523399/hazo.top`
4. 等待自動部署完成（約5分鐘）

部署成功後會自動獲得：
- ✅ 完整的電商網站
- ✅ 管理後台
- ✅ 數據庫
- ✅ SSL證書
- ✅ 自動備份

## 🌐 設置自定義域名

1. 在Railway專案中進入 "Settings" → "Domains"
2. 添加域名：`hazo.top`
3. 將以下DNS記錄添加到您的域名註冊商：

```
類型: A
名稱: @
值: [Railway提供的IP地址]
TTL: 300
```

## 📱 預設管理員賬號
